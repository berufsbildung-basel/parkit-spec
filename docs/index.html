Returns available parking spots for a given date, vehicle and full-/half-day filter. The vehicle owner is considered in the check.

Authorizations:
BasicAuth
query Parameters
date
required
string <date>
Example: date=2022-12-24

The date for which to check available parking spots

vehicle_id
required
string <uuid>
Example: vehicle_id=ccf8d1c6-f927-4e51-8de4-4d4a4f4be623

Identifies the vehicle and owner for which a reservation is intended.

half_day
boolean
Default: false
Example: half_day=true

Whether availability should be checked for a half-day reservation (false for full-day)

am
boolean
Default: false

If halfDay is true, whether a reservation is intended ante- or post-meridian (morning or afternoon)

reservations

Create a new reservation

Authorizations:
BasicAuth
Request Body schema: application/json
parking_spot_id
required
string <uuid>

Identifies the parking spot reserved during the time of this reservation.

user_id
required
string <uuid>

Identifies the user that made this reservation.

vehicle_id
required
string <uuid>

Identifies the vehicle (owned by the reservation user) that is intended to be on the reserved parking spot during the time of this reservation.

date
required
string <date>

The date on which this reservation is effective.

half_day
boolean
Default: false

Indicates whether this reservation was created as a half-day or full-day reservation.

am
boolean
Default: null

Indicates, if this reservation is a half-day reservation, whether the half day is ante- or post meridian (first, or second half of the day).

Responses

Production server

https://api.example.com/reservations

Test server

https://test-api.example.com/reservations

Development server

https://dev-api.example.com/reservations
